# Pixel Art : Salamèche

Ce projet est un script Python qui génère un dessin en Pixel Art du célèbre Pokémon **Salamèche**.

---
## 🎨 Présentation du Projet

Le programme utilise le module graphique `graphics_nsi`, fournit par mon professeur de NSI, pour reconstituer une image bloc par bloc dans une fenêtre de 300x300 pixels.

---

## 🚀 Comment lancer le projet ?

### Prérequis
* Avoir Python 3 installé sur votre machine.
* Disposer du module `graphics_nsi.py` dans le même dossier que le script `salamèche.py`.

### Exécution
Ouvrez votre console ou terminal dans le dossier du projet et exécutez la commande suivante :
```bash
python3 salamèche.py
```
